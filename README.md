# Sistema de Cadastro de Usuários 🗂️

Sistema simples de cadastro de usuários desenvolvido em Python com menu interativo no terminal.

## Funcionalidades

- Cadastrar usuário (nome, idade, e-mail)
- Listar todos os usuários cadastrados
- Buscar usuário por nome
- Menu interativo no terminal

## Como executar

1. Clone o repositório
2. Execute no terminal:
python cadastro.py

## Tecnologias

- Python 3
